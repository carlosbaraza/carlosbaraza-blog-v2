**Proyecto freelance. Angular, Node, Meteor. Aplicación web, app iOS y app Android.**

Duración: **3 meses + mantenimiento durante 2 años**.

Como parte de un nuevo emprendimiento, desarrollé una experiencia completa para un servicio de conserjería de lujo en línea. Los usuarios podían usar la app para programar tareas: llamar un taxi, recoger algo, comprar vuelos, comprar flores, etc.

- Diseño
- Aplicación web
- Aplicación iOS
- Aplicación Android
- CRM personalizado para gestionar clientes
- CMS personalizado para gestionar el tipo de tareas que los usuarios pueden solicitar, dependiendo de su suscripción.
