He notado [múltiples](https://stackoverflow.com/questions/14667713/how-to-convert-a-string-to-number-in-typescript) [preguntas](https://stackoverflow.com/questions/23437476/in-typescript-how-to-check-if-a-string-is-numeric) sobre cómo convertir un valor String a un Number usando TypeScript.

La respuesta es que puedes convertir String a Numbers de la misma manera que en JavaScript, usando `parseInt("42")`, `parseFloat("42")`, `+"42"` o `Number("42")`.

```typescript
const numberString: string = "42";

const number: number = parseInt(numberString);
const number: number = parseFloat(numberString);
const number: number = +numberString;
const number: number = Number(numberString);
```

Múltiples enfoques devuelven el mismo tipo number al analizar un string numérico
